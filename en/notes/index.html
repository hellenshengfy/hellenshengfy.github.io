4. Archaeological Database

  • PDF: Python Code

I participated in the project led by Dr. Sheng Pengfei and Dr. Zhang Menghan (Fudan University). This project aims to build a comprehensive archaeological database and seek for multidisciplinary evidence (pollen/gene/language) of possible migration of the Neolithic communities in the southeast China. I can not share the database but I would like to share some python codes I wrote for the project to collect data.
