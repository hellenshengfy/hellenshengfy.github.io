2. Archaeological database

Multidisciplinary evidence during the Neolithic period

I participated in the project led by Mr. Sheng Pengfei and Mr. Zhang Menghan (Fudan University). This project aims to build a comprehensive archaeological database and seek for pollen/gene/language evidence of possible migration in the southeast China during the Neolithic period. I am not able to share the database but I would like to share part of python code I wrote for the project to collect data from tables in pdf within seconds.
